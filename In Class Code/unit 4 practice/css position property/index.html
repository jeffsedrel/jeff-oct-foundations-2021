Static

staticis the default value. An element with the position static; is not positioned in any special way. A static element is said to be NOT postioned and an element with its position set to anything else is said to be positioned static.

Relative

Setting the top, bottom, left, and right properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.

Second Relative

Setting the top, bottom, left, and right properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.

Fixed

A fixed eleement is positioned relative to the view port, which means it always stays in the same place EVEN if the page is scrolled. As with relative the top, right, bottom and left properties are used and needed for the final position.

Nearest ancestor element to the absolutely positioned element

This element is relateively positioned. If this elemenet was positioned static its absolutely positioned child would escape and be positioned relative to the document body.
Absolute behaves like fixed except it's relative to it's nearest positioned ancestor element, instead of relative to the viewport. If an absolutely positioned element has no positioned ancestors it uses the document body, and still moves along the page scrolling. Remember a positioned element is one whose position is anything but static.

Absolute

This element is absolutely-positioned. It's relative to its parent.
